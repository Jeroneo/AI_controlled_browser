# 🤖 DeepSeek Browser Agent

A Streamlit web application that uses the DeepSeek API to autonomously control a headless Chrome browser via Selenium. Simply type a prompt in the chat (e.g., "Search for NVIDIA stock"), and the AI will generate and execute the necessary Selenium Python code to perform the action. Features a live-updating view of the browser!

## ✨ Features
* **AI-Powered Browser Automation**: Uses `deepseek-chat` to convert natural language into executable Selenium code.
* **Live Browser View**: A background periodic refresher streams the current state of the headless browser directly to the UI.
* **Execution Context**: The AI is automatically provided with the DOM context (buttons, inputs, links) to make accurate decisions.
* **Code Debugging**: Option to view the exact Python code generated by the AI before it executes.

## 📋 Prerequisites
* Docker and Docker Compose installed on your system.
* A valid DeepSeek API Key.

## 🚀 Installation & Setup (via GitHub Registry)

This project is automatically built and published to the GitHub Container Registry (GHCR) using GitHub Actions. You don't need to build the image locally!

**1. Set up your `docker-compose.yml`:**
Ensure you have the `docker-compose.yml` file in your directory configured to pull the image:
```yaml
services:
  browser-agent:
    image: ghcr.io/jeroneo/deepseek-browser-agent:latest
    container_name: deepseek-selenium-agent
    ports:
      - "8501:8501"
    shm_size: '2gb'
    restart: unless-stopped
```

**2. Pull the latest image:**
```bash
docker compose pull
```
*(Note: If your GitHub repository is private, you will need to authenticate first using `docker login ghcr.io -u YOUR_USERNAME` and a Personal Access Token).*

**3. Start the application:**
```bash
docker compose up -d
```

**4. Access the Web UI:**
Once the container is running, open your web browser and navigate to:
```text
http://localhost:8501
```

## 💻 How to Use

1. **Enter API Key**: Open the sidebar on the left and input your DeepSeek API key.
2. **Start the Browser**: Click the **"🚀 Start"** button in the sidebar. This initializes the headless Chrome session. Wait for the "Live View" to show the Google homepage.
3. **Prompt the Agent**: In the chat input box at the bottom, type a command (e.g., *"Go to wikipedia.org and search for Quantum Computing"*). 
4. **Watch it Work**: The AI will generate the Selenium code, execute it, and the Live View will automatically update to reflect the new page state. 
5. **Stop**: When you are finished, click **"🛑 Stop"** to safely close the browser session and free up memory.

## ⚠️ Notes
* **Headless Mode**: The Chrome browser runs entirely in the background (`--headless`). You will only see what is happening through the Streamlit "Live View" images.
* **Security**: The `exec()` function is used to run AI-generated code. This is restricted to specific injected variables (`driver`, `By`, `Keys`, `WebDriverWait`, `EC`, `time`), but be mindful when deploying this outside of a local/sandboxed Docker environment.
